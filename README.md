# BerberYanımda

Berber ve müşteriyi buluşturan, randevu odaklı bir uygulama.

## Başlangıç

Bu depo, fikirden ürüne ilerlerken alınan kararları `docs/` klasöründe tutar.
Ana hedef, App Store ve Google Play üzerinden indirilebilen iOS/Android uygulamasıdır. İlk web prototipi, kullanıcı akışını hızlı doğrulamak için korunur.

## Çalıştırma

```bash
npm install
npm run dev
```

Mobil uygulama için:

```bash
cd mobile
npm install
npm start
```
